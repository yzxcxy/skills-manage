## 0.13.0 - 2026-05-11

Feature release focused on marketplace imports, collection integration, platform selection, and update checks.

### Features

- Unify marketplace skill imports with collection-based central storage.
- Add marketplace file tree browsing and richer skill detail data.
- Support importing marketplace skills through the GitHub import pipeline.

### Improvements

- Improve platform selection dialogs for individual and collection installs.
- Preserve clearer platform selection state while installing skills.
- Add tests for install dialog selection behavior.

### Fixes

- Correct the application update check source to use the active release repository.

## 0.13.0 - 2026-05-11（中文）

围绕市场导入、集合集成、平台选择和更新检查的功能版本。

### 新功能

- 将市场技能导入统一到基于集合的中央存储流程。
- 新增市场技能文件树浏览和更完整的技能详情数据。
- 支持通过 GitHub 导入流程安装市场技能。

### 改进

- 优化单个技能和集合安装时的平台选择对话框。
- 安装技能时保留更清晰的平台选择状态。
- 增加安装对话框选择行为测试。

### 修复

- 修正应用更新检查来源，使用当前发布仓库。
