## 0.12.0 - 2026-05-11

Feature release focused on collection-based skill management, platform directory ownership, read-only system skills, and remote skill update checks.

### Features

- Rework skill management around collections with nested central storage.
- Add remote skill update detection and update workflow support.
- Support read-only system skills, including Codex `.system` cache skills.

### Improvements

- Simplify platform management so each platform owns only its configured directory.
- Improve centralization behavior for collection-owned skills.
- Prevent the primary scan root from incorrectly claiming skills in system subdirectories.

### Fixes

- Avoid observation lookup for installation row IDs on the skill detail page.

## 0.12.0 - 2026-05-11（中文）

围绕基于集合的技能管理、平台目录归属、只读系统技能和远程技能更新检查的功能版本。

### 新功能

- 将技能管理重构为基于集合的架构，并支持嵌套中央存储。
- 新增远程技能更新检测和更新流程支持。
- 支持只读系统技能，包括 Codex `.system` 缓存技能。

### 改进

- 简化平台管理模型，每个平台只管理自己的配置目录。
- 优化集合归属技能的中央化行为。
- 防止主扫描根目录错误认领系统子目录中的技能。

### 修复

- 修复技能详情页安装记录 ID 查询时不必要的 observation lookup。
